ココロパン

WEBサイト｜自主制作

からだにやさしい“グルテンフリーベーカリー”をテーマにした架空店舗のWebサイトを自主制作しました。 健康志向のユーザーに親しみやすく伝わるよう、やわらかい色合いと写真を活かしたデザインで、店舗の魅力が伝わる構成を意識しています。今回はデザインのみを制作対象としました。

1-4 Topページ

目的

グルテンフリーへのこだわりや、商品の魅力を視覚的にわかりやすく伝えることで、店舗への興味喚起・新規来店や購買促進につなげる。

ターゲット

健康志向・パン屋めぐりが好きな20代〜60代の女性。地元住民や観光客など、ナチュラルなライフスタイルを求める幅広い層を想定。

制作期間・ソフト

企画・デザイン：３日（XD / Illustrator / Photoshop）

構成

[ ページ ]

  • Topページ … 店舗紹介 / 「こだわり」コンテンツ
  • Line upページ … 商品ラインナップ（パンの写真・説明）
  • Shopページ … 店舗情報/Schedule(営業日カレンダー) / Contact(問い合わせフォーム)
  • Informationページ … お知らせ・イベント情報

[ ナビゲーション ]

・こだわり　・Line up　・Shop　・Scedule　・Information　・Contact

。

詳細

  • 手書き風フォントや、紙のような風合いの背景テクスチャを用いて、グルテンフリーの“自然で温かみのある世界観”を表現。
  • 配色はナチュラルなブラウンを基調に、アクセントとしてパステルブルーを使用。落ち着いた印象に清潔感をプラス。
  • コンテンツには「営業日カレンダー（Schedule）」や「最新情報（Information）」を取り入れ、来店前に必要な情報がスムーズに確認できるよう工夫。
